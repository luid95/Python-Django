Select with JQuery

You can use de $ sign to select elements from the DOM! It will feel a lot like selecting using querySelectorAll().

For example:

  • Selecting all tags:
    • $('a')
  • Selecting all elements from the class "container"
    • $('.container')
  • Selecting all elements with id "special"
    • $('#special')
